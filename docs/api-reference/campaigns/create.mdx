# Create Campaign

Creates a new campaign for an organization.

## Request Body

<ParamField body="title" type="string" required>Campaign title</ParamField>
<ParamField body="slug" type="string" required>URL-friendly identifier, unique within the org</ParamField>
<ParamField body="orgId" type="string" required>Organization ID</ParamField>
<ParamField body="type" type="string" default="donation">One of: `donation`, `peer_to_peer`, `event`, `membership`</ParamField>
<ParamField body="goalAmountCents" type="integer">Fundraising goal in cents</ParamField>
<ParamField body="suggestedAmounts" type="integer[]">Pre-set donation amounts in cents</ParamField>
<ParamField body="description" type="string">Campaign description (supports markdown)</ParamField>
<ParamField body="allowRecurring" type="boolean" default="true">Enable recurring donations</ParamField>
