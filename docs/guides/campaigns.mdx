# Campaigns

Campaigns are the core of Give. Each campaign is a fundraising initiative with its own page, goal, and donor tracking.

## Campaign Types

| Type | Use Case |
|------|----------|
| **Donation** | General fundraising, annual funds, emergency appeals |
| **Peer-to-Peer** | Supporter-driven fundraising (walkathons, birthday fundraisers) |
| **Event** | Galas, dinners, auctions with ticketing |
| **Membership** | Recurring membership programs with tiers |

## Creating a Campaign

1. Go to **Dashboard > Campaigns > Create Campaign**
2. Enter a title, description, and optional fundraising goal
3. Set suggested donation amounts
4. Choose whether to enable recurring giving
5. Upload a cover image
6. Publish when ready — your campaign goes live immediately

## Campaign Status

| Status | Meaning |
|--------|---------|
| **Draft** | Not yet visible to donors |
| **Active** | Live and accepting donations |
| **Paused** | Temporarily stopped (existing recurring gifts continue) |
| **Ended** | Campaign complete, no new donations accepted |

## Goal Tracking

Set an optional fundraising goal and your campaign page will display a live progress thermometer showing how close you are. This creates urgency and motivates donors.
