# Events

<Note>
  Event ticketing is a **Pro tier** feature, coming soon. This page describes planned functionality.
</Note>

Create ticketed events with built-in fundraising. Galas, dinners, walkathons, auctions — all managed from your Give dashboard.

## Planned Features

- Multiple ticket tiers (General, VIP, Sponsor, Complimentary)
- Custom tax-deductible amounts per tier
- In-person check-in tools
- Virtual event support
- QR code tickets
- Combined event + donation pages
- Google Maps and calendar integration
