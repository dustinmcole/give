# Donor Management

Give includes a built-in donor CRM so you can track giving history, segment your audience, and build relationships — without a separate tool.

## Donor Profiles

Every donor gets a profile that includes:
- Contact information (name, email, phone)
- Complete giving history across all campaigns
- Total lifetime giving amount
- First and most recent donation dates
- Tags for segmentation

## Tags & Segmentation

Create custom tags to organize donors:
- Major Donor, Monthly Giver, Board Member, Volunteer
- Event Attendee, Lapsed (no gift in 12+ months)
- Campaign-specific tags

Filter and search donors by any combination of tags, giving amount, date range, or campaign.

## Data Import

Migrating from another platform? Import your existing donor data via CSV. Give maps common fields automatically and lets you review before importing.

Supported migration sources:
- Givebutter, Zeffy, Give Lively, Donorbox
- Bloomerang, Network for Good, DonorPerfect
- Any platform that exports CSV

## Data Export

Export your donor data anytime. Your data is yours — we never lock it in.
