# Integrations

Give is built API-first, so it integrates with everything. Start with our native integrations or connect to 6,000+ apps via Zapier.

## API & Webhooks

Build custom integrations using the [REST API](/api-reference/overview) and [Webhooks](/integrations/webhooks). Every action in Give emits events that you can subscribe to.

## Zapier

Connect Give to 6,000+ apps without writing code. Common automations:
- New donation → Add to Google Sheet
- New donor → Create contact in Mailchimp
- Donation succeeded → Post to Slack channel
- New recurring donor → Add to HubSpot pipeline
