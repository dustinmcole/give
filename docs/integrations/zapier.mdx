# Zapier Integration

<Note>
  Zapier integration is coming soon. This page describes planned functionality.
</Note>

Connect Give to 6,000+ apps without writing code. Automate your workflows and keep your tools in sync.

## Triggers (events from Give)

| Trigger | Description |
|---------|-------------|
| New Donation | Fires when a donation is successfully processed |
| New Donor | Fires when a new donor is created |
| New Recurring Donor | Fires when a donor starts a recurring gift |
| Campaign Created | Fires when a new campaign is published |
| Donation Failed | Fires when a payment attempt fails |

## Actions (send data to Give)

| Action | Description |
|--------|-------------|
| Create Campaign | Create a new campaign |
| Create Donor | Add a donor record |
| Tag Donor | Add a tag to a donor |

## Popular Automations

- **Donation → Google Sheets**: Log every donation to a spreadsheet
- **New Donor → Mailchimp**: Add donors to your email list
- **Donation → Slack**: Post real-time donation notifications
- **Failed Payment → Email**: Alert your team about failed recurring charges
